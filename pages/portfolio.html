Band Challenge: Multipage website

This project was made during a lesson at Coder Academy where the challenge was to create a multi-page website using HTML and CSS about a Band/Group which included a discography page, bio page, Headings, Footer and Navbar linking to all other pages. This was the first time I had experimented with creating seperate styling sheets, linking them and using external fonts. I had chosen this particular group as my girlfriend enjoys watching and listening to this group, so I wanted to build and create something that someone else enjoyed as it inspired me to work on it and make it look as good as I could with my skill set at the time.

Building my Home Office Setup

This project was made during lockdown in Melbourne where I was unable to work from the office anymore and needed a work station set up at home. I built this station with a minimalist design in mind that would both be comfortable and a productive space for me to work at. It consists of a Rustic Wooden Desk, Office Chair, Laptop, 27inch Monitor, Wireless keyboard and mouse, Headset and a plant.
